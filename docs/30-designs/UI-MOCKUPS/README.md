# UI Mockups

本目录存放每个需求文档对应的一张 SVG 效果图。

## Rules

- 所有效果图使用统一平台布局：left-side、head-bar、avatar-head、主内容区和右侧状态区。
- 所有效果图遵循内容优先、学术极简、工具型体验。
- 每个 SVG 文件大小必须小于 800K。
- 生成脚本位于 [`../GENERATE-UI-MOCKUPS.py`](../GENERATE-UI-MOCKUPS.py)。
